---
title: 快速开始
description: 使用 Multica Cloud 上手，无需自部署。
---

最快的方式是使用 Multica Cloud：注册账号，连接一台本机运行时，然后把第一个 issue 分配给智能体。

## 1. 注册

打开 [multica.ai](https://multica.ai)，创建账号并进入工作区。

## 2. 安装 CLI 并启动 daemon

推荐用 Homebrew：

```bash
brew install multica-ai/tap/multica
multica setup
```

Windows 可以用 PowerShell 安装：

```powershell
irm https://raw.githubusercontent.com/multica-ai/multica/main/scripts/install.ps1 | iex
multica setup
```

`multica setup` 会打开浏览器完成登录、发现你的 workspace，并在后台启动 daemon。

## 3. 检查运行时

进入 **设置 → 运行时**。如果 daemon 正常连接，你会看到当前机器出现在运行时列表中。

## 4. 创建智能体

进入 **设置 → 智能体**，点击 **新建智能体**，选择刚连接的运行时和一个 provider（例如 Claude Code 或 Codex）。

## 5. 分配 issue

创建一个 issue，然后把它分配给新智能体。智能体会在你的运行时上执行 task，并把进度回传到 Multica。
